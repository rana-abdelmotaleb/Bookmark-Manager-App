# Bookmark Manager

A simple web application to save, manage, and visit your favorite bookmarks easily.  
Built with **HTML, CSS, Bootstrap, and JavaScript**.

---

**How to use:**

1. Open `index.html` in your browser.
2. Add your bookmarks with the form.
3. Edit, delete, or visit any bookmark with the buttons provided.
4. All data is saved automatically in your browser using localStorage.

---

**Technologies used:**  
- HTML  
- CSS  
- Bootstrap  
- JavaScript
